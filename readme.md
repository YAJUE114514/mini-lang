这是一个意图实现简单的 stack-based 函数式编程语言解释器的项目。

* 设计原则：
参照管道式程序处理（一个函数的输出等于下一个函数的输入）；
若所需参数不足，则会原样传递后续的参数。

示例：

2 3 +

栈状态：

\[] -> \[2] -> \[2, 3] -> \[5] (程序结束，输出 5)

* 语法和类型支持

要定义变量（函数），需使用 := 运算符。如：

plus2 ( Int -> Int ) := 2 +

然后输入 3 plus2, 则会输出 5.

支持递归定义：

factorial (Int -> Int) := \[0 >] \[dup 1 - factorial *] \[drop 1] if

数组：

upto :: Int -> \[Int]

upto := \[0 >] \[dup 1 - upto append] \[drop {}] if

(or use initial \{1..}, \{a..b})
